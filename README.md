# AltDaemon-Rootless

> This is a fork of AltDaemon from AltStore by rileytestut.

This fork is intended to make AltDaemon compatible with modern rootless jailbreaks like Dopamine and palera1n. This also aims to add compatibility for forks of AltStore like SideStore and LiveContainer.

You can find AltStore [here](https://github.com/altstoreio/AltStore).
